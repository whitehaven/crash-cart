# crash-cart
Two-screen web-based medical case simulator.

I figure I'll be showing this on my own computer on two monitors. Probably my laptop monitor and projector, or maybe on one monitor with my phone picking up the controller role with the server opened on LAN.

This is meant for heart failure teaching, but necessarily will comprise all cardiovascular derangements because **differentiation from other etiologies of circulatory shock** is the key skill.
